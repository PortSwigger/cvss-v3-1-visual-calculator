CVSS v3.1 Visual Calculator provides an interactive interface for calculating and visualizing CVSS v3.1 base scores directly within your penetration testing workflow. This extension eliminates the need for external calculators by offering a comprehensive visual scoring system with real-time feedback.

Features

  • Interactive visual metric selection for all eight CVSS v3.1 base metrics
  • Real-time speedometer-style risk visualization with color-coded severity levels
  • Automatic CVSS vector string generation and base score calculation
  • Dynamic severity classification (None, Low, Medium, High, Critical)

Usage

  1. Navigate to the "CVSS Calculator" tab after loading the extension
  2. Select appropriate values for each metric using the toggle buttons:
    • Attack Vector (Network, Adjacent, Local, Physical)
    • Attack Complexity (Low, High)
    • Privileges Required (None, Low, High)
    • User Interaction (None, Required)
    • Scope (Unchanged, Changed)
    • Confidentiality Impact (None, Low, High)
    • Integrity Impact (None, Low, High)
    • Availability Impact (None, Low, High)
  3. View the calculated base score and severity rating in the left panel
  4. Copy the generated CVSS vector string for documentation purposes
  5. Use the visual risk meter for stakeholder presentations and reporting
