CVSS-v3.1-Visual-Calculator is a graphical extension for Burp Suite, designed to calculate and visualize CVSS v3.1 base scores directly within your penetration testing workflow.

During penetration testing and vulnerability assessments, its common to evaluate risk using the CVSS v3.1 scoring system. This extension simplifies that process by allowing for quick and interactive calculation of CVSS scores, providing a real-time graphical risk meter, auto-generating the CVSS vector string and base score, and allowing for direct screenshots suitable for reporting. All these functionalities are in-built, eliminating the need for external calculators or manual calculation of scores.

Features

  • Visual CVSS metric selection interface
  • Dynamic CVSS vector string and score display
  • Integrated risk meter (speedometer-style visualization)
  • Color-coded severity indicators
  • Built-in CVSS v3.1 logic (offline)
  • Designed to support penetration testing reporting

Usage

To use CVSS-v3.1-Visual-Calculator, navigate to the \CVSS Calculator\ tab in Burp Suite after loading the extension. From here, you can interactively generate CVSS score and the corresponding CVSS vector string using the visual interface. The risk meter provides a graphical representation of the risk based on the calculated score. Color-coded severity indicators and the ability to take screenshots for reporting purposes are also provided for easy reporting.
